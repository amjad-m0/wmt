# نظام إدارة المستودع (WMS)

نظام إدارة مستودع كامل للمعدات الكهربائية — أوامر شغل، أوامر صيانة، استلاف، نقل داخلي، مرتجعات، وصولات PDF، وبحث كامل في السجل التاريخي.

## البدء السريع

اقرأ ملف **`SETUP.md`** — فيه كل خطوة بالتفصيل من تنصيب البرامج لحد ما النظام يشتغل.

## التقنيات المستخدمة

- **Backend:** FastAPI (Python)
- **Frontend:** Jinja2 + Bootstrap 5 RTL (عربي)
- **Database:** PostgreSQL
- **PDF:** ReportLab
- **File Storage:** محلي (dev) / Cloudflare R2 (production)

## بيانات الدخول التجريبية

بعد ما تشغّل `init_admin.py` هتستخدم اسم المستخدم وكلمة المرور اللي أدخلتهم بنفسك.
